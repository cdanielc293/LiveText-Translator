🚀 LiveText Translator
The ultimate real-time OCR translation tool for professionals and power users.

Developed by Daniel Hagbi, LiveText Translator allows you to instantly scan, translate, and hear any text on your screen with a single keystroke. Whether you're analyzing complex data, learning a new language, or navigating technical interfaces, this tool bridges the gap between seeing and understanding.

✨ Key Features
Instant Screen Snip: Hit F2 to select any area and translate it immediately.

Interactive UI: Click on any translated word to see its specific back-translation and definition.

Dual-Language Audio: Automatically speaks the original text and its translation (Powered by Google TTS).

Persistent Word Logs: Every word you translate is saved locally for future review.

Stealth Mode: Runs silently in the System Tray with zero interference.

Startup Ready: Automatically boots with Windows so it's always there when you need it.

🛠 Built With
EasyOCR: Deep Learning-based Optical Character Recognition.

Google Translator: High-accuracy neural translation.

gTTS & Pygame: Crystal clear audio synthesis.

Tkinter: Lightweight, native-performance GUI.

📦 Installation & Setup
Clone the repo:

Bash
git clone https://github.com/YourUsername/LiveTextTranslator.git
Install dependencies:

Bash
pip install -r requirements.txt
Run the script:

Bash
python LiveText-Translator.py
Pro Tip: For the best experience, compile it to an EXE using PyInstaller and run as Administrator to allow global hotkey listening.

🌍 Customize Your Experience
The project is built to be modular. Advanced users can easily modify the source code to support any language pairs supported by Google Translate (e.g., Spanish to German, Japanese to English, etc.).

We provide the engine; you choose the destination.
